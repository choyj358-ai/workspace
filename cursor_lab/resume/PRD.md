# 5. 기능 요구사항

- **5.1 레이아웃 및 네비게이션**
  - 단일 페이지(`index.html`)에서 모든 콘텐츠 확인 가능.
  - 상단 고정 내비게이션 바에서 각 섹션(내 정보, 좌우명, 기술 스킬, 자격증, 포트폴리오)로 **스크롤 이동(앵커 링크)** 가능.

- **5.2 내 정보 표시 기능**
  - 이름, 직무, 이메일, 링크(GitHub, LinkedIn 등)를 시각적으로 보기 좋게 노출.
  - 이메일 클릭 시 메일 클라이언트가 열리도록 `mailto:` 링크 지원.

- **5.3 좌우명 표시 기능**
  - 좌우명을 강조된 디자인(큰 글씨, 인용구 스타일)으로 표시.
  - 좌우명 아래에 짧은 설명 텍스트 영역.

- **5.4 기술 스킬 표시 기능**
  - 스킬 리스트 및 숙련도 뷰(바/아이콘 등).
  - 주요 핵심 역량(마이크로서비스, AI, Flutter, MVVM)은 **하이라이트** 처리.

- **5.5 자격증 표시 기능**
  - 자격증을 목록 또는 카드 형태로 출력.
  - 정렬 기준: 최신 취득 순.

- **5.6 포트폴리오 표시 기능**
  - 각 프로젝트 카드에 다음 정보 포함:
    - 프로젝트 이름
    - 간단 설명
    - 사용 기술 태그
    - 내 역할
    - 링크 버튼 (GitHub / Demo)
  - 카드 클릭 시 새 탭에서 링크 열기.
