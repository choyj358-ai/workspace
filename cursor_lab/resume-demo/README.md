## 단일 페이지 이력서 사이트

이 저장소는 **한 페이지짜리 개인 이력서 홈페이지** 템플릿입니다.  
`index.html` 하나만 브라우저에서 열어도 바로 확인할 수 있고, 라이트/다크 모드 전환도 지원합니다.

### 구성 파일

- **index.html**: 이력서 메인 페이지 (소개, 경력, 기술, 프로젝트, 학력, 기타 섹션)
- **styles.css**: 모던한 다크/라이트 테마 스타일
- **script.js**: 테마 토글 및 사용자 설정 저장 (localStorage)
- **package.json**: 간단한 정적 서버 실행용 스크립트

### 로컬에서 실행하기

1. 이 폴더에서 터미널을 열고, 정적 서버 패키지가 없다면 한 번만 설치:

   ```bash
   npm install -g serve
   ```

2. 프로젝트 루트(이 README가 있는 위치)에서:

   ```bash
   npm start
   ```

3. 브라우저에서 `http://localhost:3000` (또는 터미널에 표시되는 주소)로 접속

### 내용 수정 방법

- **이름/직무**: `index.html` 상단의 `Your Name`, `직무 / 포지션` 텍스트를 본인 정보로 변경
- **연락처/GitHub**: 헤더 부분의 이메일, 전화번호, GitHub 주소 수정
- **경력/프로젝트/학력**: 각 섹션의 더미 텍스트를 본인 이력과 프로젝트 내용으로 교체
- **마지막 업데이트 날짜**: `index.html` 하단 푸터의 날짜를 필요할 때마다 수정

필요하다면 섹션을 더 추가하거나 빼고 싶다고 말씀해 주세요. 레이아웃/스타일을 맞춰서 조정해 드릴 수 있습니다.
